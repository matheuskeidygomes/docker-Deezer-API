# Deezer API

Aplicação web criada com React.js e Docker, para consumir a API pública disponibilizada pela plataforma de músicas Deezer. Para estilização do projeto foi utilizada StyleSheet components.

# Features

- [x] Visualizar lista de músicas recebidas através da API do Deezer.
- [x] Escutar as músicas listadas.
- [x] Redirecionar para música no site do Deezer ao clicar no botão de redirecionamento.
- [x] Pesquisar música desejada na barra de pesquisa.
- [x] Adicionar músicas favoritas em lista de favoritos.
- [ ] Remover músicas favoritas da lista de favoritos.

# Pré-requisitos

Será necessário ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Docker](https://docs.docker.com/engine/install/) e [Docker-Compose](https://docs.docker.com/compose/install/). Também é aconselhável ter um editor para trabalhar com o código, como o [VSCode](https://code.visualstudio.com/).

# Rodando a aplicação

```bash

# Clone este repositório
$ git clone <https://github.com/matheuskeidygomes/docker-Deezer-API.git>

# Acesse a pasta do projeto no terminal/cmd
$ cd docker-Deezer-API

# Execute a aplicação 
$ docker-compose up

# O servidor iniciará na porta:3000 - acesse <http://localhost:3000>

```
